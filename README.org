#+TITLE: org2wiz
#+OPTIONS: ^:{}

* 插件介绍
为知笔记的一款插件，使为知支持org mode。
* 插件使用
org 文档标题必须以 ~.org~ 结尾。插入附件，后缀为 ~.org~ ，然后在为知笔记界面点击 ~org2wiz~ 。

感谢原作者 @myflocky。
